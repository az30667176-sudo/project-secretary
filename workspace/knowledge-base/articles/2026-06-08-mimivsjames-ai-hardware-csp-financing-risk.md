# AI 訂單的錢從哪裡來？AI 硬體繁榮建立在四家 CSP 的股權與債務融資上（風險提醒文）

> **MimiVsJames 第三篇入庫**（vocus.cc）。作者自述為「未來人回到現在」的**風險結構提醒文**，非操作文非崩盤預測。**這篇與 fundamental-platform §7.6/§7.7 方法論高度契合**（反身性/估值透支/循環入帳）。

## Summary

核心論點：**AI 硬體訂單是真的，但問題在「訂單的錢從哪來」**。過去 CSP（Google/Amazon/Microsoft/Meta）用**自己的現金流**蓋 AI（良性閉環），現在轉向**債務 + 股權融資**（GOOG 史上首次 $847.5 億股權融資、Meta 跟進、Amazon 發債）→ 整條供應鏈訂單建立在「借來的、發股換來的錢」的**海市蜃樓**上。四個環環相扣風險：(1) **繁榮嚴重偏科**（硬體/算力爆發，AI 應用變現遠落後 capex — MIT 研究 95% 企業 GenAI 專案對損益無貢獻）;(2) **需求高度集中 4 家 CSP**（不是分散終端，只要兩家把 capex「增速」下調就崩）;(3) **估值已預支 2027-28**（forward PE 看似便宜是因分母用未來高成長假設）;(4) **全押在利率不能升**（CPI 是壓力測試）。**股權融資比發債更該警惕**（永久稀釋 + 融資順位理論/市場時機理論/Loughran-Ritter 新股之謎都指向頂部訊號）。**反身性閉環（Soros）**：CSP 借錢發股下單 → 硬體股噴 → 估值上修 → 資本市場更願借 → CSP 下更大單;反向轉一樣快 = **戴維斯雙殺**。**只要 capex「減速」（非停止）就會出事**。1999 光纖：技術是真的，但舉債狂鋪的公司崩 80-90%。

## Key Takeaways

### 四個環環相扣的風險
1. **繁榮偏科**：賣鏟子的賺翻（GPU/ASIC/HBM/伺服器/網路/光通訊/散熱/電力），但挖金礦的（應用變現端）金子還沒挖到。**AI 使用量 ≠ 收入 ≠ 利潤 ≠ 足以覆蓋 capex/折舊/電力/融資成本的 FCF**（中間隔四道關卡，一道都還沒走完）。MIT:企業投 $300-400 億後 95% GenAI 專案對損益無可衡量貢獻;S&P：2025 年 42% 公司放棄多數 AI 專案;IBM：僅 25% AI 計畫達預期報酬
2. **需求集中 4 家 CSP**（+ OpenAI/Anthropic 但更不賺錢）：整條供應鏈邊際需求 = 4 家董事會每年拍板的 capex 預算。**過去用自己現金流（避險型）→ 大量發債（投機型）→ 連 GOOG 都發普通股（接近龐氏型）= Minsky 金融不穩定假說的危機前兆路徑**
3. **估值已預支 2027-28**：再好的產業，估值把未來兩三年預支光就是泡沫
4. **全押利率不能升**：CPI/核心 CPI 是第一次壓力測試。通膨翹頭 → Fed 被迫升息 → 同時打三處（信用利差擴大使 $1.5 兆發債成本飆 + 折現率上行砍高倍數長久期估值 + 融資水龍頭一緊閉環反轉）

### FCF 證據（變現追不上燒錢）
- **GOOG 今年 FCF 暴跌 90%**（$733 億 → ~$82 億）
- **AMZN FCF 預估轉負**（-$170 ~ -$280 億）
- Meta 同樣受壓
- 五大廠 2025 合計發 $1,210 億公司債（2020-24 常態的 4 倍），未來幾年估需 **$1.5 兆新債**;2025 capex 已吃掉營業現金流 ~94%

### 為什麼「發股」比「發債」更該警惕（理論 + 實證）
- **融資順位理論**：理性順序 = 內部現金 → 發債 → 股權擺最後。發股 = 暗示「公司認為現在股價不算便宜了」
- **新股發行之謎（Loughran-Ritter 1995）**：發新股公司發行後 5 年顯著落後同類，投資人要多投 44% 才能 5 年後拿到一樣財富;平均每年落後大盤 5-6%、持續 3-5 年
- **Baker-Wurgler**：公司傾向股價偏高時發新股;「新發行中股權占比」越高，之後整體股市報酬越低
- **當連 GOOG 都發股、整個板塊轉向股權 = 歷史上的長線頂部訊號**
- **債會還會消失，股權稀釋永久**（永遠瓜分你對未來每塊現金流的請求權）。GOOG 過去回購縮分母 → 現在反轉擴股數

### 反身性閉環（Soros）+ 兩塊加重風險 = 像 2008
- **反身性**：CSP 借錢/發股下單 → 硬體股噴 → 板塊估值上修 → 資本市場更願借 → CSP 下更大單。兩面刃，反向自我強化下跌一樣快
- **循環入帳**：NVDA 拿 $20 億（每股 $87.2）買 **CRWV** 股票 → CRWV 拿 NVDA 晶片蓋 AI 機件 → 回頭買 NVDA 晶片（收入繞一圈回自己帳上）。CRWV 接 Meta $210 億長約（到 2032）+ Jane Street $60 億 → 變漂亮 backlog → 變借錢擔保品 = **錢在自己人之間繞圈的閉環（像 2008 次貸 MBS）**
- **表外槓桿**：CRWV 用 GPU 當抵押借 $85 億（史上首筆投資級 GPU 抵押貸款，「新興資產類別」）—— 用 3-5 年折舊報廢的晶片創造投資級長期債務;Meta $270 億塞表外 SPV。**真實槓桿看資產負債表看不清**

### Forward PE 陷阱 + 戴維斯雙殺
- AI 硬體股 forward PE 看似便宜（AVGO 2026 ~41x / 2027 ~25x / 2028 ~19x、PEG 0.53;NVDA <25x + 明年 62% 成長）—— **因分母用 2027-28 盈餘，而那是「假設 CSP capex 高速成長到 2028」算出來的**
- AVGO AI 訂單 $730 億，但喊 2027 目標 >$1,000 億 → 差額多是「目標假設」非合約。**市場買的是「到 2028 一切順利」的假設**
- **只要 capex 從「年增 50%」調成「年增 20%」（仍成長、只減速）= 拐點**：(1) sell-side 下修 E（盈餘縮水）+ (2) 成長溢價收掉（倍數壓縮）= **戴維斯雙殺**。「前瞻 19 倍便宜」的股票在 capex 只是減速時就能大跌（AVGO 本季財報好仍殺 20%+，因商業模式估值調整）

### 1999 光纖 / 2008 次貸對照
- **1999 光纖**：網路是真的、光纖今天還在用，但舉債狂鋪的公司崩 80-90%、倒一票 → **趨勢終會贏，但出錢蓋的股東陪葬 = 財富重分配**
- **跟 2008 像**：互鎖閉環 + 表外槓桿 + 循環入帳;**不像**：CSP 用長天期投資級債 + 手握千億現金，無「借不到隔夜錢」流動性風險 → 破裂「不會」像 2008 幾天全停，但別誤以為「比較安全」（市場滿手 AI 股 + 槓桿 + All in + 質借的比例極高）

### 結論與建議
- 重點不是「AI 會不會發展」（一定會），是「**現在這個股價已往泡沫/風險邁進**」。趨勢真 ≠ 股價不是泡沫（可同時成立但最終面對現實）
- **「投資最危險不是買錯產業/個股，是買錯價格」**
- 長線選股偏好：**「有收過路費、有真實付費需求、自己還有現金流的買家平台」> 「估值全靠 CSP 永續燒錢撐住的純 AI 硬體股」**
- 三個觀察訊號（出現則意義不同，作者圖示未文字化）+ CPI/6-17 Fed 會議是近期關卡
- 操作紀律：高檔追要設停損;波段找修正止跌再買 + 停損;股市過熱+蟑螂吹+「送離職信」氛圍 → 分批落袋/止盈
