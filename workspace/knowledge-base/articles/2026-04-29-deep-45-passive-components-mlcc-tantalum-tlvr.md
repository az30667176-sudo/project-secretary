# AI點燃被動元件？MLCC、鉭質電容與TLVR電感是甚麼？這次不一樣？— 深入分析第45期：被動元件

## Summary

KP 深入第 45 期由付費訂閱者投票決定，鎖定一個多數投資人連電容/電感/電阻都分不清的利基領域——被動元件。文章開宗明義是 **產業地圖而非個股推薦**（除村田這種巨頭外，多為流動性低的 niche 標的，給目標價意義不大）。八章結構帶讀者走過：認識被動元件三大家族 → 三次歷史週期教訓（2017-18 缺貨 / 2019-24 漫長去庫存 / 2025-26 第三次週期）→ AI 伺服器電力闖關（54V→1V 降壓難題 + NVIDIA Vera Rubin GPU TDP 跳到 1.8-2.3 kW + 三道黃金防線：MLCC/聚合物鋁/鉭質電容）→ 高階 MLCC 護城河（堆疊到 1000-2000 層 + 日本掌握鈦酸鋇粉末命脈）→ 鉭質電容極致壟斷（KEMET 一家佔 40%+、剛果鉭礦動盪 + 12 個月內 KEMET 第三次漲價 20-30% + 國巨 Q1 毛利衝 38.1% 14 季新高）→ TLVR 電感混戰（架構是「智慧連通管」革命但純工程問題、十幾家投入、護城河淺）→ 主要受惠者地圖（國巨一站購足 / 村田 TDK 單一極致 / Vishay 美股代表 / Panasonic 失寵 / 臺慶科 + 英飛凌 TLVR 新勢力）→ 風險清單（消費電子 50% 營收基本盤 + 銀銅 + 鉭粉成本 K 型分化 + 日本鈦酸鋇出口管制 + 2027-28 新產能集中開出可能重演 2019 崩盤）。核心結論：**「這次不一樣」三大證據**——AI 是技術門檻高的結構性缺貨（質的跳升）、買方是 CSP 兩三年藍圖採購（黏性高）、賣方學乖擴產紀律嚴明（高階產線 18-24 月不可能補上 2026Q3 缺口）；但市場一致認為「這次不一樣」的時候永遠是最危險的時候——強週期性產業的「擴產→過剩→殺價」鐵律從未改變。

## Key Takeaways

### 產業地圖與三大家族
- **被動元件 = 電子世界的「煞車與避震器」**：不產生能量，只消耗/儲存/釋放能量；NVIDIA GB300 GPU 數萬美元，但旁邊電容不夠好就會降速當機（AI 伺服器功耗是傳統 5-20 倍）
- **三大家族重要性排序**：電容器（佔 40-60% 市場、AI 主力）> 電感器 > 電阻器
- **AI 引爆「雙速市場」**：高階 MLCC 因 AI 商業化急劇攀升、中低階 MLCC 因消費電子買氣低迷陷入困境

### 三次歷史週期
- **2017-18 第一次大缺貨**：村田/TDK 砍傳統 MLCC 產能轉車用高階 → 交期 50 週、現貨價飆 30 倍、國巨 2018 淨利 +583%
- **2019 崩盤**：下游恐慌性 overbooking 吹出虛假需求泡沫，國巨營收一年蒸發 40%、股價腰斬；2019-24 漫長去庫存
- **2025Q4-2026 第三次週期**：AI 伺服器點燃 5-30% 漲價潮；高階 MLCC 交期 24-32 週、鉭質電容部分料號 40 週

### 「這次不一樣」三大證據（vs 2018）
- **質的跳升**：2018 是所有標準品一起缺貨；2026 是雙速市場，AI 需要極端高溫高壓低 ESR 高階 MLCC 與聚合物鉭質電容，技術門檻極高的結構性缺貨
- **買方變了**：2018 是手機/PC 廠（景氣波動、砍單極快）；2026 是 CSP 雲端巨頭按 2-3 年 AI 基礎設施藍圖採購、簽長期協議
- **賣方學乖了**：被 2019 嚇怕，現在以「去瓶頸」微調產能而非新工廠；高階 MLCC 產線 18-24 月才能量產 → **2026Q3 之前供給缺口不可能填補**

### NVIDIA Vera Rubin (2026H2 出貨) 的供電革命
- TDP 從 Blackwell 1.2-1.4 kW 跳到 **1.8-2.3 kW**；每顆 GPU 配 400 焦耳的 6 倍能量儲存設計
- **三道黃金防線**：(1) **MLCC** 最靠近 GPU 處理高頻突波（Vera Rubin 刻意增加超高容值用量分攤風險）；(2) **聚合物鋁固態電容** 中頻濾波主力（但 NVIDIA 為避免單一供應商斷鏈，新設計微調比例）；(3) **鉭質電容** 高溫高壓下不縮水的最後保險
- **沒有公司能贏者全拿**：MLCC = 精密陶瓷、聚合物鋁 = 電化學表面處理、鉭質 = 危險化學冶金 + 稀有金屬，三種基因完全隔離 → 這是國巨豪擲千金併購 KEMET 的根本原因

### 高階 MLCC 護城河
- **堆疊微米級千層派**：低階只疊幾十層，高階必須在同樣厚度疊 1000-2000 層，每層 < 1 微米，混入一顆灰塵就短路 → 頂級 vs 二線分水嶺
- **日本掐住「陶瓷心臟」**：鈦酸鋇粉末頂級量產技術幾乎全在日本（Sakai Chemical / Nippon Chemical / Fuji Titanium），村田自己提煉；中國再砸錢蓋廠仍須跟日本買「麵粉」
- **MLCC 三層分化**：
  - 頂層 (~15% 規模，>40% 利潤)：村田/三星電機/太陽誘電寡佔 90%+，是真正賣方市場
  - 中段 (~45%)：台廠 (國巨/華新科) 戰場 — 日韓大廠戰略放手中段標準品的「借來的時間」
  - 底層 (~40%，<15% 利潤)：中國風華 + 三環 600 億人民幣擴產，2024-25 ASP 持續下跌
- **「擠壓效應」沒發生** 因為池子根本不相通：日韓搬走的是頂層產線，從設備到製程都做不了 0805 通用品

### 稼動率 + 庫存指標：尚未進入恐慌性搶貨
- 2026 Q1 三星電機稼動率 92%、村田 95%（接近滿載）
- 三星電機庫存天數降至 30 天（業界健康水位 40 天）
- **目前連「重複下單」爭議都還沒出現** — 對比 2018「交期 50 週、現貨價飆 30 倍」，當前缺貨潮還沒進入恐慌性搶貨；MLCC 廠商獲利預期還有上修空間
- **2026Q3 = 真正燃點**：Vera Rubin 量產加速 + IT 旺季雙重需求疊加，最有可能引爆「米缸真的沒米」的恐慌性下單

### 鉭質電容 — 集齊「技術無可取代 + 原料被掐住 + 極端壟斷」
- **三巨頭寡佔**：KEMET（國巨收購，全球 40%+）+ AVX（京瓷收購）+ Vishay 合計 60-70% 市佔
- **連續漲價驗證壟斷**：KEMET 12 個月內第三次漲價（2026/4/1 鎖定 T523/T520/T521/T530 聚合物鉭質系列），每次 20-30%；Kyocera AVX 跟進 15-30%；交期從 8-10 週拉到 18-40 週
- **國巨 2026 Q1 毛利率 38.1% 創 14 季新高、年增 2.5pp**，明確歸功於鉭質連續漲價 + AI 需求激增；漲價幅度甚至「超過」原料成本漲幅 = 從 AI 硬體設計獲取超額價值
- **剛果鉭礦動盪**：50%+ 全球鉭礦來自剛果民主共和國；2025 初礦災 + 叛軍把持，2026 Q1 歐洲鉭粉報價飆破 $500/公斤，今年以來 +90%；鉭精礦同步 +40%
- **天花板隱憂**：2025 全球鉭質電容市場僅 $13.4 億、2030 預估 $16.7 億 (4.51% CAGR) — 投資邏輯「高利潤率、中等成長率」，價值不在規模而在綁定客戶
- **「設計繞過」風險**：歷史上 Apple/Samsung 曾把手機鉭電容換回 MLCC；當前三星電機已量產 220μF/330μF 超高容、TDK 1000 層多層化讓微型 MLCC 接近鉭質水準；若鉭質被取代，預算回流 → 村田/Samsung/TDK 雙重受惠

### TLVR 電感 — 智慧連通管革命，但護城河淺
- **AI 晶片極端 di/dt**：前一秒待機、下一秒生影片需 1000A 超大電流；傳統各自獨立電感反應不及
- **TLVR 架構**：在主機板底下牽額外線把所有閥門串聯，瞬間互助；供電效率高 + 反應快 + 還能省電容空間
- **成本炸彈雙夾擊**：銀價飆破 $121/盎司 (年增 3 倍) + 銅價突破 $14,500/公噸；村田/TDK 2026 準備調漲
- **護城河淺**：難點在「磁性材料配方 + 繞線設計」是純工程問題，不需挖剛果稀有礦也不需奈米陶瓷燒結；TDK/Eaton/Infineon 國際大廠 + 臺慶科/順絡電子等十幾家已投入 → 進入門檻低、定價權隱憂
- **比例尺視角**：2026 全球 TLVR 市場僅 $1.5 億、2031 仍只佔整體電感 5-6%；AI 伺服器中 TLVR 也僅佔電感用量 20%、其餘 80% 仍傳統架構 → **TLVR 是頂端小眾，整體電感漲價 25-40% 才是基本盤**

### 主要受惠者地圖
- **國巨 (Yageo)** — 唯一一站購足（MLCC + 鉭質 + 聚合物三種電容都齊），KEMET 併購是關鍵；MLCC 是現金流、鉭質才是估值錨；2026 Q1 AI 營收占比 14-15%，指引全年突破 15%
- **村田 + TDK** — 單一領域技術極限領導者；村田掌控鈦酸鋇粉末配方到燒結全鏈；高 P/E 評價合理
- **Vishay (NYSE: VSH)** — 美股鉭質電容指標股；整體毛利 19.6% 受半導體業務拖累但鉭質訂單動能強
- **Panasonic + Nichicon** — 聚合物鋁電容遭 NVIDIA 微調份額擠壓；絕對量仍大但成長爆發力不如鉭質/MLCC
- **臺慶科 + 英飛凌** — TLVR 新勢力，臺慶科 AI 伺服器營收占比 ~22%（純度最高）+ 英飛凌業界首款 TLVR 四相功率模組；長期定價權有限

### 催化劑時間軸
- **2026 Q2 (現在)**：村田 4/1 起調漲鐵氧體磁珠/功率電感/RF 電感/共模扼流圈；觀察台廠 (國巨/華新科) Q2 法說會跟進
- **2026 Q3 = 真正燃點**：Vera Rubin 量產 + IT 旺季疊加；村田下一輪漲價 + 大中華稼動率破 90% 雙條件同時觸發 → MLCC 全面上行
- **2026 Q4**：若 Q3 引爆缺貨，Q4 法說將出現「重複下單」疑雲；客戶端庫存天數衝破 50 天即警覺
- **2027 Q1**：村田新產能驗收，若需求疲軟時開出 = **2019 崩盤劇本重演**
- **2027-2028**：NVIDIA 800V HVDC 架構導入決定下一波電源類被動元件需求拐點

### 風險清單
- **AI 變現速度不如預期** — 整個被動元件多頭最大前提
- **消費電子 50-60% 營收基本盤萎縮**：手機/PC 仍是基本盤，AI 成長必須先「填補」消費電子衰退；低階 MLCC 庫存無法去化即被迫降價清庫存
- **成本面 K 型分化**：銀漿佔 MLCC 成本僅 ~10%（BME 製程已用鎳）vs 繞線電感銅 30-45% + 銀漿 15-25% + 積層片式電感銀漿 30-50% → **真正打擊電感廠不是 MLCC 廠**；鉭質跟鉭粉走、跟銀銅脫鉤
- **日本鈦酸鋇粉末出口管制情境**：若日本仿半導體光阻劑劇本管制，最受害是中國風華/宇陽；村田/Samsung 護城河更深；國巨兩頭擠壓
- **2027-28 新產能集中開出**：村田 + 三星電機 + 中國各廠新產能同期上線，若需求走軟 = 價格崩跌

### 投資人應追蹤的領先指標
- **村田/TDK 月營收 (每月 10 號)**：任一連 2 月 YoY < 5% → 警示消費電子拖累擴大
- **台廠 MLCC 庫存天數**：從目前 110-120 天升至 140 天+ → 警示需求疲弱
- **四大 CSP 資本支出指引**：任一下修下年指引 > 10% → 警示 AI 投資週期見頂
- **MLCC + 鉭質報價 (TrendForce 月度)**：高階 MLCC 環比連 3 月下跌 → 警示供需反轉
- **NVIDIA/AMD 新世代 GPU 公告**：明確降低被動元件用量 → 警示長期需求結構改變

### 結語：保持敬畏
- 強週期性產業的「擴產→過剩→殺價」鐵律從未改變
- 2026 特殊性是真實的（AI 真實需求 + 產能紀律 + 雙速市場），但不能讓被動元件變「永遠成長股」
- **市場一致認為「這次不一樣」的時候永遠是最危險的時候**
