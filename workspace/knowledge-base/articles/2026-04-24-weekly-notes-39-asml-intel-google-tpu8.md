# 台積電不買ASML新機？Intel皇者歸來？- KP思考筆記(第39期)

## Summary

KP 在費城半導體指數**連漲 18 天**（罕見極端情緒區）後提醒：別猜頂部，回歸選對標的。引用「悲觀者聽起來聰明，賺錢的通常是樂觀者」框定本期六主題。**主題一 ASML High-NA**：張曉強再次明確表態 A16/A14/**連 A13（2029 量產）都不用** High-NA EUV，把市場最後一絲幻想掐熄。原因：機台 4 億歐元（舊款 2 倍）+ 曝光區域只有一半（大晶片要拼接）+ 單次曝光成本是 low-NA 的 2-2.5 倍 → 用低 NA + multi-patterning 反而更划算。台積電「讓 Intel 去踩坑」+ 把 4 億歐元 capex 轉投先進封裝（CoWoS/SoIC）邊際效益更高。短期 ASML 仍靠 low-NA 飆量（2026 出貨 60 台、2027 拉到 80 台），但 2030 年 600 億歐元的 high-NA 故事要打問號 → multiple 該 de-rate。最深層意義：**摩爾定律從「誰做出最小電晶體誰贏」轉向「誰用最低成本做出夠用電晶體誰贏」**——經濟可行性取代純技術領先。**主題二 ASTS**：Blue Origin 第三次發射故障 BlueBird 7 衛星損失（保險全 cover、生產線沒停 — 8-10 號 5 月中出貨、目標每月 6 顆 + 多元發射夥伴 SpaceX/印度 LVM3/Blue Origin），但兩天後 4/21 拿到 **FCC SCS（Supplemental Coverage from Space）正式商業執照**——全美只有 Starlink 和 ASTS 兩家有，等於拿到 4 兆美元市場的金鑰匙。直接解鎖 Verizon 剩 4,500 萬美元預付款 + FirstNet 里程碑付款 + 海外電信商 MOU 升級正式合約。風險質變：從「監管能不能過」變成「執行力 + 現金 burn」，年底前能否 45 顆衛星進軌 + 第一通商業電話。**主題三 Google TPU 8**：4/22 揭幕，雙版本（**TPU 8t Sunfish 訓練 / TPU 8i Zebrafish 推理 + Agentic**）。最大驚喜不是 121 exaFLOPS 等表面 spec，而是 **記憶體大躍進**：TPU 8t 每顆 216 GB HBM、Superpod 2 PB 共享；TPU 8i 每顆 288 GB HBM、SRAM 是上代 3 倍、整 pod HBM 從 49 TB 跳到 **331 TB**——HBM 超級週期再加柴。供應鏈四夥伴而非兩家：**Broadcom 全包高階訓練（合約至 2031，毛利高）+ MediaTek 輔助 + Google 自下場買晶圓記憶體做推理（成本 -20%~-30%）+ Marvell MPU + Intel Axion CPU**。關鍵 mindset shift：Google 從「晶片設計者」變「系統定義者」。**主題四 Intel**：CEO 陳立武「一年前討論能否活下來，今天討論能多快擴產能」。Q1 營收 $13.6B（vs 預期 $12.4B）、Adj EPS $0.29（vs 預期 $0.01，**29 倍**）、DCAI $5.1B YoY +22%；CPU 復甦兩個原因 = 供應鏈瓶頸解決 + AI 從訓練轉推理 / Agent 讓 CPU 重新成為調度層。Foundry 營收 $5.4B QoQ +20% 但外部客戶才 $1.74 億、營業虧 $24 億——關鍵在 **18A/Intel 4/Intel 3 良率「優於預期」、18A 從原 2026 年底目標提前 3-6 個月到年中達成**。**14A 是真正決戰**（首個大規模用 High-NA + 第二代 RibbonFET + PowerDirect 背面供電），如果拿不到大型外部客戶可能取消。**Elon Musk 4/22 宣布 TeraFab 採用 Intel 14A**——第一個重量級外部客戶。先進封裝營收上修從「數億」到「每年數十億」，馬來西亞檳城後端 2027 貢獻營收。但別過度興奮：Foundry 仍虧 $24B、外部客戶 $1.74 億；**敘事從「能否做到」轉「何時做到」、股價從「便宜」轉「樂觀」**。**主題五 ServiceNow**：營收 $3.77B 超預期、訂閱 +22% 超預期、AI 營收目標從 $10 億上修至 $15 億 (+50%)、上調全年指引——**但股價暴跌**。三個原因：(1) 中東 / 伊朗衝突拖延 on-prem 大單，cRPO 略低於預期；(2) 史上最大併購 Armis（網安 $77.5 億）短期壓利潤率，全年 op margin 指引從 32% 下調至 31.5%、Q2 只有 26.5% (vs 預期 30%)；(3) **AI 焦慮才是真重點**——Q2 有機 cRPO 指引 17-18% vs Q1 22% 在減速，扣掉 Armis + 地緣，核心成長可能只剩十幾趴；席位制 → 消費/成果制定價轉型「看起來像被迫防守」+ 續約率微降至 97%；當增長要靠併購（Armis $77 億）+ 併購還稀釋 margin → 質疑放大。Bill McDermott 強調 22 年數據優勢 + 950 億工作流 + 7 兆筆交易，但市場要「證明」不要「我們相信」。SaaS 全行業困境：OpenAI/Anthropic 直接切入應用層，老牌廠每代典範轉移都說「我不一樣」——市場選擇先不信、等數據累積。**主題六 JPMorgan 私募信貸**：Jamie Dimon 公開警告「契約鬆散、估值不透明、PIK 利息滾雪球、像看到一隻蟑螂背後可能很多」，4/23 摩根大通資產管理部門宣布**大舉進軍私募信貸**——典型 Dimon 式「公開警告私下進場」。核心邏輯：用內部商業銀行體系**直接產出貸款**（origination），自家客戶經理已有 deal flow，不用滿世界搶案子，成本低 + 風控透明 + 與 Blackstone/Apollo 等差異化。時機點：散戶資金（~$2,500 億）的「半流動性」基金陷贖回危機（贖回申請達 40% 但每季只能 5%），但機構資金（80% of $2T 市場）穩定但擁擠——JPM 帶著主權基金 / 養老金長線資本進場搶機構份額。組織上放在固定收益部門而非另類投資 = 暗示公開債券和私募信貸界線正模糊。3 月已收緊對外部私募信貸基金放貸條件（「別人的 risk」做風險管理）+ 4 月擴大「自己的」業務。私募信貸第三波：銀行帶資產負債表 + 放貸通路 + 機構客戶關係重新進場，從「神秘高收益避風港」回歸「信貸審核 + 資金成本」金融本質。

## Key Takeaways

### 市場 / 心法
- 費半連漲 18 天進入極端情緒區，但「猜頂部」是隨機博弈無參考價值；應專注「選對標的 × 拉長時間維度 × 深化產業認知」
- 「悲觀者聽起來聰明，賺錢的通常是樂觀者」(Pessimists sound smart, but optimists make money)
- 兩年前 NVDA 被嫌貴，今天回頭看當時高點已是地平線起點 → 持有優質底層資產 + 給夠時間，價值會透過成長體現

### Topic 1 — ASML High-NA：護國神山把最後一絲幻想掐熄
- 張曉強這次把話講死：A16 / A14 / **連 A13（2029 量產）都不用 High-NA**，市場那一絲「台積電會改主意」的希望被正式掐熄
- High-NA 經濟性算不過來：機台 4 億歐元（舊款 2 倍）+ 曝光區域只有一半（大晶片要拼接增加風險）+ 單次曝光成本是 low-NA 的 2-2.5 倍
- 台積電「讓 Intel 去踩坑」策略：Intel 是 High-NA EUV 全球首位客戶 14A 上陣 → TSMC 用成熟便宜 low-NA 大賺 AI 晶片錢，等 2030+ High-NA 成熟再進場
- 台積電的本質是 **資本紀律**：4 億歐元 × 10 台 = 40 億歐元寧可拿去擴 CoWoS / SoIC 先進封裝，邊際效益高很多
- ASML 短期靠「量」(low-NA): 2026 出貨 60 台 + 2027 80 台供不應求；中期高毛利的 high-NA 故事被砍 → 2030 年 600 億歐元目標要打問號 → 估值 multiple de-rate
- **最深層啟示：摩爾定律戰場在變** — 從「誰做出更小電晶體誰贏」變「誰用最低成本做夠用電晶體誰贏」。當單一節點 R&D > $200 億 + 一台機 4 億歐元 + 設計一顆 3nm 要 $5 億 → **經濟可行性取代純技術領先**
- 開放問題：Intel 押 High-NA 的錢，2030 後是「先發制人」還是「白繳學費」？

### Topic 2 — AST SpaceMobile：掉一顆衛星卻贏了整個美國
- BlueBird 7 在 Blue Origin 第三次發射中損失 — **比想像輕很多**：(A) 衛星有保險現金成本近零；(B) 德州工廠 8-10 號 5 月中出貨、產線跑到 32 號、目標每月 6 顆；(C) 多元發射夥伴 SpaceX/印度 LVM3/Blue Origin
- 兩天後 4/21 **FCC SCS 商業執照下來** — 全美只有 Starlink 和 ASTS 兩家有；可合法用 AT&T / Verizon 黃金低頻直連普通手機
- 解鎖：Verizon 剩 $4,500 萬預付款 + FirstNet 里程碑 + 海外電信商 MOU → 正式合約。等於拿到 **$4 兆美元市場金鑰匙**
- ASTS 真實護城河：(1) **垂直整合產能**（每月做 6 顆 → 一顆損失邊際成本越來越低）+ (2) **監管授權**（牌照才是真威力）
- 風險質變：FCC 前的風險是「監管過得了嗎」，FCC 後變成「執行力 + 現金 burn 多快」+ 年底前 45 顆衛星進軌 + 第一通商業電話
- Blue Origin 失敗 ≠ 戰略錯：第一節 booster 完美重複使用著陸（追了好幾年的里程碑）；參考 SpaceX Falcon 1 前三次都炸 — 火箭這行不摔幾次走不到量產

### Topic 3 — Google TPU 8：從晶片設計者到系統定義者
- 雙版本：**TPU 8t (Sunfish, 訓練)** Superpod 9,600 顆 = 121 exaFLOPS / **TPU 8i (Zebrafish, 推理 + Agentic)** 全新 Boardfly 拓撲降低同步延遲
- 真正驚喜不在算力，**在記憶體**（一腳踹穿 memory wall）：
  - TPU 8t 每顆 **216 GB HBM**、Superpod 2 PB 共享記憶體
  - TPU 8i 每顆 **288 GB HBM**、晶片內 SRAM 是上代 3 倍、整 pod HBM 從 49 TB 跳到 **331 TB**
- 這是「Agent 應用一個請求要處理過去 chatbot 20-50 倍資料」的直接 response → **HBM 超級週期再加一把柴**
- 供應鏈不是 2 家是 4 家：**Broadcom（全包高階訓練 8t、合約至 2031、不可替代 SerDes + 封裝）+ MediaTek（輔助封裝 8i）+ Google 自買晶圓和記憶體 → 推理成本 -20% ~ -30% + Marvell（MPU）+ Intel（Axion CPU + 基礎設施）**
- 連鎖效應：(A) 單位成本降 → Cloud 毛利率升；(B) 單一供應商風險稀釋；(C) TSMC 2nm 仍是瓶頸但 Google 多家夥伴搶 = 議價籌碼
- **Mindset shift：Google 完成從「晶片設計者」到「系統定義者」蛻變** — 透過記憶體極致堆料 + 供應鏈博弈築起難跨越的成本與技術門檻

### Topic 4 — Intel 皇者歸來？
- CEO 陳立武 framing：「一年前討論能不能活下來，今天討論能多快擴產能」
- Q1 巨幅 beat：營收 $13.6B (vs $12.4B 預期) / **Adj EPS $0.29 vs $0.01 預期 = 29 倍超出**
- DCAI $5.1B (YoY +22% 比預期高 ~$7 億) — 兩個推力：(1) 上季伺服器 CPU 供不應求現在供應鏈理順 + (2) **AI 從訓練轉推理 / Agent → CPU 重新成調度層 + 控制平面**
- 前瞻指引：Q2 DCAI 雙位數環比 + 全年伺服器 CPU 出貨雙位數成長 + 動能延續至 2027
- Foundry 數字仍弱（營收 $5.4B QoQ +20% 但外部客戶才 $1.74 億、營業虧 $24 億）但 **Intel 4 / 3 / 18A 良率「優於預期」、18A 提前 3-6 個月（從原 2026 年底目標到年中達成）** → 帶動毛利率改善是虧損收窄主因
- **14A 是真正決戰**（第一個大規模用 High-NA EUV + 第二代 RibbonFET + PowerDirect 背面供電 + 預期效能功耗比 +15-20% / 密度 +30%）— 沒有大型外部客戶可能取消
- **Elon Musk 4/22 宣布 TeraFab 採用 Intel 14A** = 第一個重量級外部客戶 + 全世界最高調的科技公司之一押注
- 14A 同階段良率/效能/PDK 進度比 18A 還好；風險試產 2027 / 量產 2028-29 / 早期設計承諾從 2026 H2 開始
- **先進封裝（被低估的近期引擎）**：CFO 上修潛在營收從「數億」到「每年數十億」；馬來西亞檳城後端設施 2027 貢獻營收；EMIB / Foveros 全球只有 3 家能做 Intel 是其中之一
- 領導力驗證：陳立武打通組織官僚（決策從「幾月」到「幾週」）+ 引用 Andy Grove「只有偏執狂才能生存」
- **估值張力**：敘事從「能不能做到」轉「什麼時候做到」，但股價從「便宜」轉「樂觀」。Foundry 還虧 $24B / 一季、外部客戶 $1.74 億。任何良率 / 訂單 / 營收稍微不如預期回調幅度可能劇烈

### Topic 5 — ServiceNow 暴跌反映什麼
- 看似正面：營收 $3.77B 超預期、訂閱 +22% 超預期、AI 營收目標 $10B → $15B (+50%)、上調全年指引
- **股價暴跌三原因**：
  - 地緣政治：中東 / 伊朗拖延 on-prem 大單 → cRPO 略低預期（拖累 ~0.75pp）
  - 併購：史上最大 Armis 併購 $77.5 億 → 全年 op margin 指引從 32% → 31.5%、Q2 只有 26.5% vs 預期 30%
  - **AI 焦慮才是真重點**：Q2 有機 cRPO 指引 17-18% vs Q1 22% 在減速；扣掉 Armis + 地緣，核心成長可能只剩十幾趴
- 戰略上正確的轉型卻像「被迫防守」：席位制 → 消費 / 成果制（為了 hedge「AI 代理取代人工」恐懼），但短期毛利受壓 + 續約率微降至 97%
- 當增長要靠併購維持 + 併購還稀釋利潤率 → 質疑被放大
- 管理層 push back：Now Assist 年合約 > $100 萬客戶 YoY +130%、多產品交易 +70%、AI Control Tower 翻倍、>50% 新業務是非席位制
- McDermott：22 年數據優勢 + 950 億訓練過工作流 + 7 兆筆交易 + 亞秒級處理 + Context Engine + AI Control Tower → 嘲諷對手 AI 是「客廳小把戲 (parlor tricks)」缺乏治理與編排
- **核心問題：市場要「證明」不要「我們相信」** — 即便業績超預期 + 管理層信心爆棚，股價仍可暴跌
- SaaS 全行業困境：OpenAI / Anthropic 直接切入應用層；每代典範轉移老牌都說「我們不一樣」（雲端、行動、現在 AI），市場選擇先不信、等連續 3-4 季執行驗證
- 投資最難不是判斷公司好不好，是判斷**市場什麼時候會相信它好**

### Topic 6 — JPMorgan 警告私募信貸卻自己進場
- Dimon 公開警告：信用標準鬆散、契約形同虛設、估值不透明、PIK 利息滾雪球；「看到一隻蟑螂時很可能已經有很多隻」
- 4/23 JPM Asset Management **大舉進軍私募信貸**（典型 Dimon 式：公開警告 + 私下磨刀）
- 過去憋屈：2016 拆出 HPS 後眼睜睜看對方賺翻、2024 收購 Monroe Capital 談判中途撤離
- **核心商業邏輯**：用內部商業銀行體系**直接產出貸款** (origination)，自家客戶經理就有 deal flow → 不用滿世界搶案子 + 成本低 + 風控透明
- 時機：散戶資金（~$2,500 億 of $2T 市場）的「半流動性」基金陷贖回危機（申請 40% 但每季只能贖 5%）→ 機構資金（80% market）穩定但擁擠
- JPM 帶**長線機構資金**（主權基金、養老金）進場 — 不會季度 mark-to-market 跑掉
- **不公平優勢**：自家商業銀行每天跟中型企業打交道，這是 Blackstone / Ares / Blue Owl 拿不到的 deal flow
- 組織安排細節：放在 **固定收益部門** 而非另類投資 → 暗示公開債和私募信貸界線正模糊
- Dimon 真正的擔心是「別人」（鬆散標準的玩家 + 把半流動性產品賣給散戶的基金），不是 JPM 自己（機構資金 + 自家風控）
- 3 月先收緊對外部私募信貸基金放貸條件 + 4 月擴大自己業務 = 對「別人的」做風險管理 + 開「自己的」生意
- **私募信貸第三波**開啟：純基金狂飆 → 散戶湧入 → 銀行帶資產負債表 + 放貸通路 + 機構客戶關係重新進場
- 過去幾年被包裝成「神秘高收益避風港」，現在回歸金融本質：**信貸審核 + 資金成本**
