# 費半大跌但它們抗跌：半導體「晶片股」vs「設備股」—— Rubin 驗證、2027 WFE 重估

> **MimiVsJames 第二篇入庫**（vocus.cc）。本篇承接其半導體系列，講設備股輪動邏輯。

## Summary

論證費半/QQQ 大跌中**設備股（AMAT/ASML/LRCX/KLAC）逆勢創高**的訊號 = 市場在 AI 產業內**重新選擇佈局**，從「晶片股」輪到「設備股」。核心邏輯差異：**晶片股交易「成長能否再超預期」**（門檻被推太高，連基本面仍成長的 AVGO 都因「沒再大幅超預期」而估值壓縮）;**設備股交易「2027 年的產能建設」**（晶圓廠擴產領先終端出貨 6-9 個月到 1-2 年）。**設備股不需賭哪種晶片架構贏 —— GPU/ASIC/CPU 都要過同樣有限的先進製程/EUV/沉積/蝕刻/量測/HBM/封裝產能**（賣鏟人邏輯）。2027 WFE 被大投行整體上修（三大引擎：先進邏輯 + HBM/先進 DRAM + 先進封裝），把設備業營收/EPS/估值中樞全部上修+後移。Rubin 量產的意義**不是帶來短期設備訂單，而是驗證需求可信度 → 降低 2027 產能過建/訂單取消的疑慮**。資金從設備龍頭擴散到**二階供應鏈（UCTT/ICHR/MKSI/NPO）**—— 低基期 + 高營運槓桿（每增 1 美元營收，更高比例轉成 EPS）。

## Key Takeaways

### 為什麼設備股逆勢抗跌（兩種定價邏輯）
- **晶片股**：交易「下一季營收能否再上修、財報能否再超預期、估值能否被快速消化」。門檻被推太高 → **AVGO 是代表**（基本面仍成長但沒再大幅超預期 → 估值壓縮）
- **設備股**：交易「為了 2027 年有足夠晶片可賣，晶圓廠現在必須增加多少設備」。從決定擴產→無塵室→下單→製造→安裝→驗證→量產 = 跨多季到 6-9 月+ → **股價提前反映未來 6-9 月訂單預期**
- **關鍵優勢**：設備商**不需判斷哪種晶片架構贏** —— GPU/ASIC/AI CPU 都競爭，但都要過有限的先進製程/EUV/沉積/蝕刻/量測/HBM/封裝產能 → 「選所有贏家都必須用的設備」優於「選贏家」

### 2027 WFE 上修 = 三大資本支出引擎同時擴張
1. **先進邏輯**：3nm→2nm + FinFET→GAA（改變電晶體結構，增加沉積/蝕刻/材料工程/量測步驟）→ 晶圓產能↑ + 每單位產能設備價值↑
2. **HBM / 先進 DRAM（最被低估的新增變數）**：過去假設記憶體廠只「轉換」既有 DRAM 產能做 HBM（對設備無增量）;**新情境 = HBM 消耗先進 DRAM 晶圓 + 一般 DRAM 需求沒消失 → 總產能問題 → 必須新建無塵室/晶圓/設備**。HBM 比一般 DRAM 更耗晶圓面積 + TSV/減薄/鍵合/封測。DRAM 往先進節點 → EUV 在記憶體重要性提高
3. **先進封裝 / 3D 異質整合**：AI 晶片競爭從先進製程延伸到封裝（HBM 堆疊/Chiplet/互連密度/TSV/大尺寸封裝）→ AMAT 預估先進封裝收入高速成長

### 大投行 2027 WFE 上修數字
- Cantor Fitzgerald：2026 $1,450 億 / 2027 $1,850 億 / 2029 $2,500 億
- Morgan Stanley：2026 $1,490 億 / 2027 $1,910 億 / 2028 $2,150 億
- Barclays（最積極）：2027 由 $1,590 億大幅上修到 **$2,095 億**（高於當時市場 $1,720 億）
- ASML 產能上限上修：JPM/MS 重估 Low-NA EUV 年產能可從假設 90 台拉到 110 台+（不蓋新廠）→ **供給端營收天花板也被抬高**

### Rubin 的真正意義（對設備股）
- **不是**七月出貨帶來設備訂單（首批 Rubin 晶圓設備早已交付安裝驗證）
- **是降低 2027 設備訂單被取消/縮減/延後的風險** —— 驗證下一代 AI 平台從路線圖進入實際量產（先進製程良率跨過工程驗證 + HBM4 供應鏈規模化爬坡 + 先進封裝進入產能爬坡）
- 在 Rubin 確認前，機構把 2027 WFE 上修視為 sell-side 過度樂觀;確認後 = 有終端產品/實際訂單/供應鏈 Build 作驗證 → 設備週期高點**後移**

### 設備龍頭（交易需求能見度）
- **ASML**：曝光設備稀缺性（EUV）
- **AMAT**：材料 + 製程步驟增加 + 前段 + 先進封裝同時擴張（認為 2026 半導體設備 +30%、先進封裝 +50%）
- **LRCX**：蝕刻/沉積強度提升（記憶體 + 3D 結構更複雜）
- **KLAC**：晶片/封裝更複雜 → 不能容忍低良率 → 更多量測/檢測/製程控制
- 四檔一起創高 = 市場交易的是「整個晶圓製造資本支出基準上調」，非單一設備/製程。從純週期股 → 「結構成長 + 週期彈性」

### 二階供應鏈（交易 Build Plan 傳導 + 營運槓桿）
- **AMAT 最大客戶開始提供滾動 8 季需求預測** → AMAT 提高 Build Plan/庫存/物流 → 二階供應商可為 2027 提前備產能（過去能見度只有 1 季-半年）
- 二階特性：市值小、客戶集中、毛利率低、**前一輪低潮基期低** → 訂單回來時**低基期營運槓桿**（營收 +15% 但 EPS 從近零放大）
- 個股：
  - **UCTT**：設備模組 + 外包製造 + 清洗/塗層。半導體純度高、與 AMAT/LRCX 高連動 = **設備產量週期高 Beta**。風險:毛利率低、庫存先於營收
  - **ICHR**：氣體/流體輸送系統。製程越先進流體控制越複雜 → 受惠量↑ + 單機價值↑。**2026 Q1 營收 $2.236億→$2.561億(+15%)、GAAP OM -6.2%→+0.8%、EPS $0.01→$0.15** = 低基期營運槓桿教科書
  - **MKSI**：平台型（真空/電源/電漿/雷射/運動控制/量測），客戶分散、毛利率較好，但半導體只佔部分 → EPS 彈性不如 UCTT/ICHR
  - **NPO（新增推薦）**：前段子系統 + 腔體精密零件 + 清洗/塗層/翻修 + 光學濾光片。連結 AMAT/ASML + 售後市場特性（裝機基數↑→服務需求↑）。半導體純度較低（含工業/航太/能源）→ 股價較穩、波段向上

### 選股思維轉變
「從選擇『贏家』→ 選擇『所有贏家都必須使用的設備』」。設備龍頭 = 長期價值上修;二階供應鏈 = 獲利斜率變陡（重估更接近交易）

- **這篇給了我 momentum 篩選撈到的高動能半導體設備名單「故事」**：我上次「動能 × Library」篩選的 top 60 裡，**UCTT（mom 99.6, +434%）、ICHR（mom 97.7, +376%）、LRCX（mom 98.2, +327%）、AMAT（mom 97.3, +237%）** 都在前列 —— 現在 MimiVsJames 提供了完整 thesis。**「動能 + 故事」交集確認 = 這條設備鏈**
- **fundamental-platform ER 候選（設備鏈，全新主題軸）**：
  - 龍頭：**LRCX / AMAT / KLAC**（ASML 是 ADR/荷蘭）—— 交易 2027 WFE 需求能見度，deep-30 也覆蓋
  - 二階：**UCTT / ICHR / MKSI / NPO** —— 低基期 + 營運槓桿，但 momentum 已極端（UCTT +434%）→ ⚠️ §7.6 「動能 + 故事 = 可能已搶跑」**很可能 Hold-lean-Sell**（如我跑的 ON 案例）
- **⚠️ 關鍵估值警示（連結今天的 meta-pattern）**：這些設備/二階股**動能極端**（UCTT +434%、ICHR +376%）= 跟 MU/MRVL/ON 同病。MimiVsJames 看的是「2027 獲利重估」（earnings 重估），但若**股價已搶跑 earnings**（像 ON）→ Hold-lean-Sell。**跑 ER 時關鍵 = 比較「2027 EPS 上修速度 vs 股價漲幅」**（HPE/DELL 模式 = Buy;ON 模式 = Sell）
- **HBM → DRAM WFE 連結 = 我 MU ER 的補強**：本文「HBM 消耗先進 DRAM 晶圓 → 一般 DRAM 收緊 → 記憶體廠必須增總產能 → 新增 WFE」= 我 MU/SNDK 記憶體超級週期的**設備端外溢**。MU 的需求 = LRCX/AMAT/KLAC 記憶體設備的訂單
- **§7.5 賣鏟人 + §7.8 變現能見度**：「選所有贏家都必須用的設備」= §7.5 賣鏟人;「AMAT 滾動 8 季需求預測 + Build Plan」= §7.8 變現能見度頂級。設備鏈是 §7.8 最乾淨的應用場
