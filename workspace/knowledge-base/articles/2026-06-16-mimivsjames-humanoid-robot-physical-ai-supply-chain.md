# 人形機器人概念股怎麼選？2026 下半年 Physical AI 供應鏈最值得投資的個股（上篇）

> **新作者入庫**：MimiVsJames（vocus.cc，量化工程師、美國大型資管公司）—— 非 FOMO 研究院。風格：聚焦黑科技/小眾精密供應鏈 + 直白口語。**本篇為上、下篇之上篇**（下篇講量產瓶頸）。

## Summary

論證 2026 下半年 AI 資金的**第二條主線 = 人形機器人 / Physical AI**（第一條仍是 Rubin 量產）。核心論點：人形機器人的投資重點**不在 AI 模型「會不會說人話」，而在「能不能長時間穩定工作」** —— 決勝在**致動 + 傳動系統**（關節）的精密製造，不在雲端算力。**最先受惠的不是整機品牌（Tesla/Figure），而是無法繞過、技術最難取代的精密零組件供應鏈**（一旦拿到量產訂單，營收/估值彈性比整機商更早反映）。下半年催化劑密集：NVDA SIGGRAPH（7/19-23 展 Physical AI）→ Optimus V3 + Fremont 產線（7 月底-8 月）→ 世界機器人大會（8 月）。資金面：SPCX IPO 後高 Beta/事件驅動資金尋找下一個故事 → 可能轉向 Optimus 人形機器人。**選股聚焦兩個最難量產的核心零件**（行星滾柱絲槓 + 諧波減速機），推薦 4 檔美股：**PH（大型）+ RRX / MOG.A / CGNX（中小型，機器人營收比重高）**。

## Key Takeaways

### 兩條下半年投資主線
1. **Rubin 量產**（AI 大腦算力升級）—— 吸引重訂單/獲利的機構資金，伺服器/網路/光通訊/電力/散熱
2. **人形機器人**（Physical AI，替 AI 裝眼睛/神經/肌肉/四肢）—— 短期盈利能見度不如 Rubin，但**估值彈性 + 股價想像空間更大**。7 月中下旬-8 月吸引事件驅動 + 提前布局資金

### 為什麼難量產 = 不在 AI，在傳統工業製造工藝極限
- 人形機器人 = 控制閉環：**感測環境 → AI 判斷 → 關節動作 → 感測結果 → 即時修正**。任一環節延遲/誤差/過熱 → 任務失敗甚至安全風險
- 機器人要進入「為人類設計的工廠/倉庫」面對地面落差、物品位置改變、重量差異、旁邊有人 → 比固定式工業機械手臂難太多
- 作者犀利點：連大語言模型在電腦前都常出錯（「叫他拿番茄醬可能理解成拿槍」），走進物理世界安全疑慮更大 → **致動/傳動的可靠度才是真瓶頸**

### 三大系統拆解
1. **感知/控制/Physical AI（大腦+神經）**：外部感知（RGB/立體視覺/光達）+ 本體感知（IMU/編碼器/電流感測）+ 接觸感知（六軸力矩/觸覺）。平台玩家:NVDA GR00T/Isaac/Cosmos（模型/模擬/運算）;ADI/ALGM/NOVT（感測/回饋/控制）
2. **致動 + 傳動（肌肉）= 成本最高、最難量產**：整合式關節致動器 = 馬達 + 減速機/絲槓 + 軸承 + 編碼器 + 感測器 + 驅動器 + 煞車散熱。物理瓶頸（彼此衝突）:同時要高扭力 + 低重量 + 低背隙 + 高效率 + 低噪音 + 低發熱 + 長壽命
3. **電力系統**：有限重量/體積下同時驅動數十關節 + 續航

### 四個關鍵致動零件
- 無框力矩馬達（關節肌肉，輕量化）/ 諧波減速機（手腕手肘小型精細低背隙）/ RV 擺線減速機（腰髖大力承重）/ 行星滾柱絲槓（旋轉轉直線推拉，膝髖手肘）

### ⭐ 兩個最難取代的零件（量產瓶頸核心）
1. **行星滾柱絲槓**：線性致動核心（雙腿推進/重物提舉=股四頭肌）。**公差需微米級** → 須極罕見昂貴的「高精度螺紋磨床」（產能多在歐洲/日本少數設備廠，交期 1-2 年）= 最嚴重擴產瓶頸
2. **諧波減速機**：手腕/手指/手肘（強迫薄金屬柔輪彈性形變咬合剛輪）。鋼材配方不純或熱處理差 1 度 → 數百小時後金屬疲勞斷裂。**冶金學 + 材料科學極致工藝，需數十年數據累積，砸錢/聘工程師難複製**

### 推薦個股（4 檔）
- **PH（大型，市值 >$700億）**：全球運動控制巨頭，航太/軍工累積極深液壓/機電/線性致動。工業產線所需耐高壓閥門/高階氣動夾爪/特製線性滑軌，**美股最穩定有量產能力的大廠**
- **RRX（Regal Rexnord，中小）**：子公司 **Kollmorgen（科爾摩根）** = 全球**無框扭矩馬達前三**、美股最強純玩家。無框馬達體積最小、瞬間爆發扭力最大 = 隱形冠軍
- **MOG.A（Moog，中小）**：軍工級致動（五代戰機/洲際飛彈精密伺服閥 + 動態控制致動器）。**高頻響精密控制 + 耐衝擊結構（平衡）** —— 少數能用在高階商用人形機器人動態平衡的硬體商
- **CGNX（Cognex，中小）**：工業機械視覺（硬體相機 + 邊緣識別演算法）= 機器人智慧眼睛。3D 視覺感測不受環境光干擾

### 投資紀律（作者強調）
- 大型股 PH（基本面利潤支撐）;中小型 RRX/MOG.A/CGNX（機器人營收比重高，Tesla/Figure 量產訂單砸來時營收+估值彈性最直接）
- **題材市場還沒注意** → 除 PH 外其他三檔「可關注，確切進場點作者會再講」
- 警告：不要追「只靠影片展示」的概念股（會跳舞的是耍猴戲），要找**量產時無法繞過、技術最難取代**的零件商
- 機器人是買來「工作/操」的（每年舉 50kg×1000 次、走 10 公里、用 5 年），精密耐用度要求極高;EV 火燒車 → 未來機器人自燃燒屋的新聞會出現 → 可靠度=護城河

## 投資 implication（接 fundamental-platform）

- **解釋了 user 為何跑 PH ER**（2026-06-16）—— PH 是本文人形機器人大型股首選。但**注意定位差異**:本文看 PH 的「液壓/線性致動/氣動夾爪」機器人角度;我的 PH ER 主驅動是**航太週期**（aero backlog $8B/OM 30.2%），機器人/液冷是小 kicker。**PH 的人形機器人曝險目前佔比小，是長線選擇權非主戲**（與本文「大型股基本面支撐」一致 = 不是純機器人 play）
- **3 檔新 fundamental-platform ER 候選（人形機器人精密供應鏈，全新主題軸）**:
  - **RRX (Regal Rexnord)** — Kollmorgen 無框馬達，純度最高的機器人致動 play
  - **MOG.A (Moog)** — 軍工級致動 + 動態平衡，高純度
  - **CGNX (Cognex)** — 機器視覺，機器人眼睛
  - 這三檔「機器人營收比重高」= 比 PH 純,符合「便宜/被低估 + 真催化劑」的 Buy 候選 pattern（但需跑 pipeline 驗證估值是否已搶跑）
- **方法論連結**:本文的「最先受惠是供應鏈不是整機品牌」= 我的 §7.5「賣鏟人」邏輯（淘金熱賣牛仔褲）;「砸錢/聘工程師無法複製的極致工藝護城河」= §7.7 narrative monopoly 的實體版（行星滾柱絲槓/諧波減速機的工藝壟斷）
