## Summary

MacroMicro 21 頁研究報告，以「超越循環」框架重新定位 2026 年 AI 投資機會：表面看起來是供應鏈漲價，實際上是 Agentic AI 觸發的「結構性升級」+ 台積電 Q1 法說確認的「Megatrend 已成立」+ 美伊戰爭測試後的「斷供風險可控」。五大主題：(1) Claude Code 帶動 Agentic AI 落地，IDC 預測 2029 全球 AI agents 超過 10 億，Token 消耗推升半導體銷售近 50%、淡化 3-4 年庫存循環；(2) 拆解兩條漲價路徑（直接受惠 vs 產能排擠），記憶體 / ABF / 散熱（水冷板）三產業共同擺脫消費電子週期，TrendForce 把 Server DRAM ASP 從 +60~65% 上調至 +93~98%、PC DRAM 上調至 +110~115%；(3) 荷姆茲海峽斷供風險拆解 — AI 供應鏈 60~99% 走空運所以海運衝擊低，五項中東關鍵材料逐一評估，氦氣（卡達 1/3）+ 溴（以色列 35%）風險最高但有 2-6 個月緩衝；(4) 台積電 Q1 2026 全面超預期（毛利率 66.2% 創新高、AI CAGR 上調至 56-59%）+ N3 首次打破歷史策略追加產能（亞利桑那二廠 / 日本熊本二廠都導入 N3）；(5) 生產力循環三階段追蹤架構，對比 PC / 手機 / EV / AI 四代科技週期，判斷「AI 當前在階段二、邁向階段三」，並用 100+ 圖表指標分「基建財務 + 模型動態」兩大模塊持續追蹤；CSP 廠資本支出比例 — 傳統大廠 <30%、Neocloud >100% — 標出 2026-27 將是 AI 落地變現的關鍵檢驗期。

## Key Takeaways

- **Agentic AI 是分水嶺**：黃仁勳 2026 GTC 把 Claude Code 定位為首個能「真正執行工作」的代理型 AI（讀檔 / 調用工具 / 寫程式 / 編譯 / 測試 / 迭代），算力需求從「會說話」進入下一波「會做事」拐點
- **Claude 滲透率急升**：Ramp 企業採用率數據顯示 Claude 自 2025 起加速，是各模型中成長最強；IDC 預測 2029 全球活躍 AI agents > 10 億；Claude Code 全面上線錨點 = 2025/2/24
- **代理生態降門檻**：Cowork + 開源平台 OpenClaw（俗稱龍蝦）降低個人/企業導入門檻；Token 消耗量級躍升，推升半導體銷售近 50%
- **AI 淡化半導體景氣循環**：傳統 3-4 年庫存循環因 Agentic 需求進入第二波上升週期（自 2025 起），不再單純跟著消費電子起伏
- **漲價路徑兩條**：(a) 直接受惠 — 新科技導致新需求 → ASP 上升 → 為應對需求增產 → 市佔提升；(b) 間接受惠 — 產能轉換到新科技 → 原產線供給下降 → ASP 上升 → 世代更替後市佔下降；最終結構升級、總規模擴大
- **TrendForce 2026Q1 報價大幅上調**：Server DRAM ASP +93~98%（前估 +60~65%）、eSSD +75~80%（前 +33~38%）、TLC/QLC NAND +85~90%；間接受惠的 PC DRAM 更猛 +110~115%、Mobile DRAM +58~63%
- **AI 將佔 DRAM 晶圓產能過半**：SemiAnalysis 預估 2026 年 AI 佔全球 DRAM 晶圓總產能正式超越非 AI — 記憶體產業已逐步擺脫「消費性電子」泥沼
- **ABF / 散熱複製記憶體腳本**：ABF 從 NB/PC/手機驅動轉向 GPU 直接受惠（下半年反映在營收）；散熱隨 AI 晶片 TDP 上升 → 全水冷 → 從「系統級」轉「封裝級」剛需 → 水冷板（Cold Plate）廠商受惠
- **庫存仍健康**：記憶體 / ABF / 散熱三產業存貨天數均落於低檔（ABF 穩定 45-50 天），未大幅攀升 — 表示供給尚未追上需求，漲價有延續性
- **荷姆茲海峽風險低於市場擔憂**：AI 供應鏈不依賴海運 — DRAM/NAND >99% 空運、AI 伺服器/GPU 模組 >67% 空運、智慧型手機 ~77% 空運、筆電 ~60% 空運；海運主要是面板 / CCL/銅箔大宗 / 桌機/印表機
- **中東關鍵材料風險地圖**：氦氣（高，卡達佔全球 1/3，但 TSMC + Samsung 有 2-5 個月安全庫存 + 80-90% 回收 → 半年以上緩衝）；溴（高，以色列 35% + 約旦 15%，但可長期儲存）；氫氣（中，多元來源含 SMR）；氖氣（低，烏克蘭/中國/美國）；石腦油（低，多元來源）— 整體斷供機率不高，主要影響成本傳導
- **TSMC Q1 2026 全面超預期**：營收 358.9 億 USD（共識 346-358）、毛利率 66.2%（共識 63-65%）、營業利益率 58.1%（共識 54-56%）、EPS NT$22.08（預估 NT$20.87）；資本支出落區間高標 520-560 億
- **Q2 指引再升級**：營收 390-402 億（vs 共識 384）、毛利率 65.6-67.5%（vs 共識 64%）— 強勁需求驗證 AI Megatrend 不是短期現象
- **AI 加速器晶片 CAGR 上修**：2024-29 從 50% → 56-59%（mid-to-high 50s）；推動力是 Agentic AI 帶來的 Token 消耗大幅跳升 → 推升先進製程晶片 + CoWoS 需求
- **N3 史上首次追加產能**：台積電打破歷史策略，把部分台灣 N5 機台轉成支援 N3，且亞利桑那二廠 / 日本熊本二廠都計畫導入 N3（分別 2027 H2 / 2028 量產）— 過去先進製程都「先在台量產 → 才考慮海外」，這次外廠直接吃 N3 是 AI 需求強到逼迫策略改變
- **海外廠時程提前**：日本熊本二廠 N3/N6 從原訂 2029 H1 提前到 2028；亞利桑那三廠 N2/A14/A16 量產 2027；德國德勒斯登 N28/22 等特殊製程 2027 — 整體擴張節奏比之前指引更快
- **生產力循環三階段框架**：硬體開發突破（基建）→ 軟體應用加速 → 軟硬結合再加速；對比 PC（1970-1990s）/ 手機（2000s-）/ EV（2010s-）/ AI（2017-future）四代週期；判斷 AI 目前在階段二、邁向階段三（Agentic AI + 終端應用爆發）
- **CSP 廠 vs Neocloud 風險分歧**：傳統 CSP 大廠資本支出/營收 < 30%（本業穩定有護城河）；Neocloud > 100%（高槓桿擴張）— 2026-27 將是 AI 落地變現的關鍵檢驗期，風險開始顯現
- **AI Agent 資安疑慮升溫**：AI 接觸企業敏感數據後，S&P 500 電話會議中「資安相關」字眼提及次數顯著攀升；Coding Agent 帶動 Git Pushes 飆升，金融業 Agent 滲透率自 2025 底加速
- **追蹤指標 100+ 筆**：MacroMicro 整理「重點供應鏈財務數據」+「模型應用動態」兩大模塊；庫存是判斷漲價能延續到何時的關鍵 — 當供給追上需求 → 庫存堆積 → 漲價結束
