# 為何800V HVDC是必然？重電、輕電與散熱缺一不可？- 深入分析第50期：HVDC與台達電

## Summary

本文論證 **800V HVDC（高壓直流電）是 AI 資料中心的必然演進**，核心戰場從傳統電網的「三千公里」壓縮到機櫃內的「三公尺」。當單櫃功耗從傳統 10kW 衝到 NVIDIA Blackwell 120kW、下世代 Rubin Ultra 600kW（= 400 個家庭用電），傳統 AC 配電到物理極限：48V 下 600kW 需 12,500A 電流 → 銅纜粗到塞不進機架。這是 140 年電流戰爭的反轉 — 愛迪生 DC 當年因「無法變壓」輸給特斯拉 AC，如今 SiC/GaN 功率半導體解決了升降壓難題，DC 重新勝出。文章用 **重電（灰區）/ 輕電（白區）/ 散熱（液冷）三足鼎立、缺一不可** 的框架，論證能同時跨足三道門檻的玩家極稀有，並指出 **台達電（Delta）是 HVDC 隱形主角** — 唯一同時練了 50 年電源 + 散熱、且用 SST（固態變壓器）從零件層往上攻進系統層的異類。引述 SemiAnalysis 報告框架，定位為產業深度研究（不談個股估值/買賣）。

## Key Takeaways（核心論點）

### 物理必然性
- **問題本質**：600kW/櫃，48V 配電需 12,500A → 銅線粗如大腿、塞不進機架 → 必須拉高電壓到 800V 降電流（發熱損耗 ∝ 電流平方）
- **歷史平行**：愛迪生 DC 1890s 輸 AC 是因當時只有變壓器能升降壓、而變壓器需 AC；今日 **SiC/GaN** 讓 DC 可半導體式升降壓 →「愛迪生沒錯，只是早生 140 年」
- **空間反轉**：過去 HVDC = 800kV/3000km 跨洲輸電；現在 HVDC = 800V/3 公尺 機櫃內
- **抄電動車作業**：EV 為超快充已從 400V→800V，上游 SiC 功率元件 + 車規接頭已量產成熟便宜 → 資料中心直接享紅利

### 重電 vs 輕電（第二章框架）
- **灰區（重電）**：高壓變壓器/UPS/配電盤，「萬伏級洪水」，鋼鐵與銅線的物理世界 → ABB/施耐德/西門子主場。**佔空間但不產營收（成本）**
- **白區（輕電）**：機櫃內 50V→0.8V，「低電壓、超大電流、零容錯」，IC Design 功力 → 台達電/光寶主場。**唯一產營收空間（黃金地段）**
- **失衡的灰白比**：傳統 1:1，AI 時代灰區重電設備暴脹 → 蠶食白區 GPU 空間。**800V HVDC 的核心商業目的 = 縮小重電體積、把空間還給白區**
- **AC 折返跑損耗**：每次 AC↔DC 轉換損 2-5%，500MW 廠累計 **25MW 白白浪費**

### PUE 死穴
- 1000 櫃 × 600kW = 600MW IT；PUE 1.1 → 浪費 60MW；800V 壓到 1.05 → 省 **30MW**
- 30MW = 每年省 **~$2,500 萬電費**（工業電價 $0.09/度）+ 釋放配額多塞 50 個 600kW 機櫃
- 「PUE 小數點後第二位 = 百億級企業護城河」

### 三階段演進（引 SemiAnalysis 框架）
1. **2024-25 機櫃內整流**：480V AC 拉到櫃後，櫃內 PSU 整流 → 佔 GPU 空間 + 廢熱
2. **2026-27 Sidecar（過渡）**：整流/BBU 移到旁邊獨立側櫃，1 櫃供 1-4 運算櫃 → **台達電/Vertiv/Schneider 核心戰場**。不需打掉重建舊機房
3. **2027 後 DC-native（終局）**：新建機房入口直接 SST 把中壓 AC → 800V DC 當骨幹，BBU 直掛母排，省掉 UPS 的 AC↔DC 折返。**SST 成熟度 + 成本是關鍵**
- **兩條路雙軌並存**：舊機房升級到 Sidecar 為止；新建超大案才一步到位 DC-native。贏家是能同供兩種架構者

### 電熱雙生子（第五章 — 為何談電必談熱）
- **破除誤區**：散熱 90%+ 是為 GPU（120kW 櫃中電源廢熱僅 ~6kW）→ HVDC 省的那點熱是零頭
- **真正綁定有三重**：
  1. **共用同一條水管**：SiC 功率模組/母排也熱到要液冷 → 與 GPU 共用冷卻液/CDU/幹管，同一個熱預算
  2. **溫度↔效率惡性循環**：SiC 導通電阻隨溫升 1.5-2x → 散熱差→更熱→損耗大→更熱
  3. **「濕的 800V 直流電」致命安全**：DC 不過零點 → 電弧不自滅；冷卻液若漏到帶電母排 = 複合風險 → 必須電+熱同隊設計（不整合會出人命）
- **終極大獎**：SiC 耐熱 → 可用 30-45°C **溫水冷卻** → 拔掉最耗電的冷氣壓縮機。因果鏈：用 HVDC → 選 SiC → 系統耐熱 → 溫水冷卻 → 關冷氣

### 規格暗戰
- **Diablo 400**（CSP 開源，±400V 雙極，較安全、沿用 EV 元件） vs **NVIDIA 單極 800V**（一條線 800V，銅省效率高但絕緣/防電弧嚴苛）
- 本質是「買家定標準 vs 賣家挾晶片話語權」的主導權爭奪，但兩者都指向 HVDC

## 台達電投資論點（第六-八章 — 重點）

### 為何是台達電
- **50 年雙基因**：鄭崇華「電源教父」（電視變壓器→PC SPS→伺服器 PSU→HVDC）+ 全球最大散熱風扇商之一 → **電與熱兩面都練 50 年的極少數玩家**
- **白區原住民往上攻**：靠 SiC 把變壓器從「物理機械」變「半導體裝置」→ **SST（固態變壓器）是攻進重電的攻城槌**，中壓直轉 800V DC，台達電估效率拉升近 5%
- **Grid-to-Chip 一站式 Power Rack**：SST→Sidecar→HVDC 母排→DC/DC PSU→BBU→液冷，整櫃出廠驗證 + 單一保固窗口 → 賣「電與熱在同櫃相安無事」
- **NVIDIA 參考設計關鍵角色**：GB200 世代 PSU/BBU 已是官方標配；次世代與 NVIDIA 並肩制定電力+散熱規範 → 預先寫進微軟/Meta/AWS 採購清單

### NVIDIA 三層生態地圖
1. **晶片層**（ADI/Infineon/MPS/Navitas/安森美/ROHM/ST/TI）= SiC/GaN 燃料供應商，是台達電供應商非對手
2. **電源零件模組層**（**台達電**/Flex/光寶/Lead Wealth/麥格米特）= 把晶片做成模組
3. **機房系統層**（Eaton/Schneider/Vertiv/ABB/西門子）= 供電骨幹
- **台達電的異類性**：被歸在零件層，卻用 SST 往上爬系統層 + 緊貼晶片層 → 唯一「跨三層整合者」

### 競爭格局
- **系統層**：重電巨頭（施耐德/伊頓/ABB/西門子）根基深但缺晶片肌肉記憶，靠收購補洞（施耐德 2024 收 Motivair 液冷）；Vertiv 機房電熱整合強（過去兩年股價爆發主因）但 SST/中壓直轉天生空白
- **零件層**：Flex 靠併購拼湊、光寶守本業缺重電上游、中國廠做零件 → **台達電是唯一原生整合 + 往上跨者**
- **華為**：PowerPOD 全棧原生能力三道門檻一道不缺，是除台達電外最接近全棧者 — 但**被地緣政治擋在 NVIDIA 三層之外**（「強到不被允許上桌」）
- **結論**：三道門檻原生布局最完整 = 台達電 + 華為；能進歐美超大規模供應鏈的（扣掉華為）= **台達電領先**

### 終局思辨（第八章）
- **最危險的敵人是客戶不是對手**：NVIDIA + 雲端巨頭「自己的東西自己做」垂直整合趨勢（如蘋果收編晶片）。但短期難取代 — (1) 速度+量產規模鴻溝 (2) NVIDIA 參考架構鎖定 (3) **800V+液冷的安全保固責任** 需實體工業廠扛包票
- **護城河是動態的**：需持續用技術領先 + 執行力捍衛
- **真正分野在 2027 後**：Sidecar 過渡期比「部署速度」（Vertiv 甜蜜點）；DC-native 時代比「中壓直轉 + 電熱原生整合」硬實力 → **台達電放大效應 2027 後才顯現**，握的是「有時效的優勢牌」
- **範式轉移**：摩爾定律失效 → AI 算力靠「規模堆疊」→ 資料中心是「巨型熱力學機器」→ 競爭維度從「算力效率」轉到「**能量效率（每度電能算多少次）**」→ 解釋微軟買核電廠、亞馬遜蓋廠在發電廠旁
