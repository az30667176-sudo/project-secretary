# 5 大巨頭財報，所有你要知道的事情 - KP思考筆記(第40期)

> KP 方法論：不「看圖說故事」（Google 漲=好 Meta 跌=壞 是看後視鏡開車），不堆數據（AI 時代隨手問 Claude 就有）。**穿透數字看商業動機 + 敘事位移**。真正 Alpha 藏在 AI 暫時讀不懂的「未來方向微小訊號」（電話會議語氣 + narrative 是否位移）。

## 主題一：Meta — 廣告機器碰上資本支出無底洞

- **「想像力 vs 帳單」博弈**：Q1 營收 +33%（廣告 2021 來最快）漂亮,但市場不買帳因 **Capex 上調到 $1,250-1,450 億**（連續幾季上調,組件/記憶體變貴 + AI 算力黑洞）→「誠信危機」FCF 壓縮
- **AI 確實在賺錢（但只在廣告優化,無新產品線）** — 三大自研系統：
  - **GEM**（廣告中央大腦,LLM 架構基礎模型,上帝視角）
  - **Lattice**（統一框架,數百獨立小模型整合,跨 Feed/Reels/Stories 通用語言,降運算成本）
  - **Adaptive Ranking Model**（VIP 總管,低轉化用小模型,高轉化才調昂貴 GEM → 因材施教省算力）
  - 成果:登陸頁轉化率 +6% + 主版位 +1.6% → 廣告單價 +12% → 解釋 33% 營收。Reels 觀看 +10%,WhatsApp/Messenger 商業 AI 週對話 >1000 萬（年初 10 倍）
- **致命弱點：沒給 ROI 數字**（回收期長 + 回報間接 + 成本上升快）。商業 AI 變現「未來應該會有」非「現在有」。Manus 收購($20億)被中國政府攔胡 = 地緣風險
- **本質是「對 Zuckerberg 意志力的信仰測試」**：他偏執 + 絕對控制權,會一直燒到贏或燒光。「要不要持有 Meta = 你信不信 Zuckerberg 搞得起 AI」。讀者投票 34% 放空/避開(最多)
- ⚠️ Reality Labs 累虧 >$830 億

## 主題二：Amazon — AWS 再加速，雲端一哥發力

- **AWS 過去幾年最好**：營收 $376 億(+28%,15 季最快),營業利益 $142 億(OM 37.7%),年化跑速 $1,500 億。Q4→Q1 史上最大季增
- **AI 拉動核心雲端支出（被低估的關鍵）**：「AI 支出與核心成長強相關」—— AI 工作負載需大量 S3/資料庫/網路（後訓練/RL/agent 都用核心服務）→ 自我強化飛輪（更多 AI 生產→更多核心支出→更高黏著→更多 AI 遷入）
- **自研晶片變結構性護城河**：Graviton+Trainium+Nitro 年化跑速 >$200 億(+三位數),若對外銷售近 $500 億 = **全球前三大資料中心晶片業務**。OpenAI 承諾 2 GW Trainium(2027 爬坡) + Anthropic 5 GW + 十年 >$1,000 億 AWS 支出（Amazon 回投 Anthropic 高達 $250 億）。Graviton 比 x86 性價比高 40%。暗示未來向第三方賣整機架
- **可見性比市場意識高**：**RPO backlog $3,640 億**（上季 $2,440 億,還不含 Anthropic $1,000 億+）= 過去 12 月營收 2.7 倍 → 去風險化 capex（2026 $2,000 億支出對應 2027-28 變現）
- **OpenAI 進 Bedrock**（微軟獨家鬆綁後）→ AWS 定位中立多模型平台
- 營業利潤率創紀錄 13.1% = 高質成長。Jassy 內部:AWS 2036 可達 $6,000 億年化（之前預期 $3,000 億）

## 主題三：Microsoft — 重演 AI 的「Windows 時刻」

- 從「輕資產軟體」轉「重工業 AI 公用事業」（計算單位變「吉瓦」,產能受限）
- **「扣除 OpenAI」真相**（4/27 合作調整:微軟不再付營收分成,OpenAI 付 20% 有上限且與 AGI 脫鉤,IP 授權延至 2032 但非獨家）→ 強調獨立敘事:商業訂單「扣除 OpenAI +7%」(含則 -4%);RPO「扣除 +26%」(含 +99% 被膨脹)。「我們成長不需綁定 OpenAI」
- **Copilot 進展不是座位數是使用深度**：付費座位 >2,000 萬（季增 33%）,每用戶查詢量環比 +20%,週參與度與 Outlook 相當,月活年初至今 6 倍,>50,000 座位公司翻 4 倍。滲透率 ~4-5%（4.5 億商業 M365 座位）
- **AI 年化營收跑速 $370 億(+123%)**（首次高調揭露,涵蓋 Azure AI/Copilot/GitHub/Security）
- **從助手到代理**：90% 財富 500 有活躍代理;Copilot Cowork 跨應用多步驟長任務 = 數位員工。用內部 17 EB 數據訓練（公司集體記憶）
- **從「賣座位」到「席位+使用量 (Seats + Usage)」**：GitHub Copilot 2026/6/1 全面轉 usage-based;Dynamics 365 客服近 60% 買 usage 額度。= 巨額 capex 與營收建立彈性聯繫（賣電力）

## 主題四：Google — 完美財報擊碎 AI 質疑

- **回答兩年最大疑問「AI 錢有沒有賺回來」= 有,且快得多**
- **雲端爆發**：Google Cloud $200 億(+63%,遠超預期 48-50%,首破 $200 億)。**企業 AI 首次成主要成長驅動**;生成式 AI 產品(Gemini) **+800%**;Gemini Enterprise 付費月活季增 40%（2,800 公司/800 萬席位）。**Cloud OM 17.8%→32.9% 幾乎翻倍**。運算受限（有產能營收會更高）
- **搜尋沒死反被 AI 增強**：搜尋營收 $604 億(+19%),查詢量創新高。AI Mode/AI Overviews 讓用戶更常回來
- **全棧優勢**（獨一無二同時控制每層）：TPU(第八代) + Gemini + Vertex/Agent Platform + 搜尋/YouTube/Android 分發 + 3.5 億付費訂閱 → 飛輪
- **TPU 開始對外銷售（關鍵策略轉變）**：第八代 TPU(8t 訓練/8i 推論)直接交付外部客戶裝在客戶資料中心 = 直接挑戰 NVIDIA。2027 營收「有意義」
- **RPO backlog $4,620 億**（上季 ~$2,300 億翻倍,**首次超過 AWS $3,640 億**,一半 24 月內轉營收）
- **Capex 上調 $1,800-1,900 億**（2027「顯著更高」）但市場正面接受（backlog 給信心 + capex 雙重目的:滿足雲需求 + 建 TPU 庫存）

## 主題五：四巨頭分歧與真相

- **2026 四巨頭 capex 合計 $6,500-7,250 億**（Alphabet 預告 2027 顯著更高）= 科技史最大集中投資
- **共識：AI 基建是生存門檻**（從 2022-24「紀律/效率」語言 → 「這是我們時代最關鍵投資」,投資不足風險 > 投資過度）。Azure +40% / AWS +28% / Google Cloud +63% 證明需求真實加速 = 緩解泡沫擔憂
- **分歧：市場用「變現能見度」區分贏家輸家**：
  - **Google = 最大贏家**（$4,620 億 backlog = 白紙黑字確定性）
  - **Meta = 最慘**（capex 上調但 AI 用於內部黑箱無外部收入 + Reality Labs 虧損）
  - **MSFT/AMZN = 中間**（穩健但「僅微幅超預期」/ AWS 王者歸來但市場評估 $2,000 億 capex 能否帶超額回報）
  - **「並非所有 AI capex 在投資者眼中平等」**
- **進入「證明給我看」階段**：CEO 們從「我們認為會有回報」→「我們有信心會有回報」（記錄在案的承諾,若落後變負債）
- **能源/供應鏈現實**：Meta 明說記憶體漲價是 capex 上調驅動;都追核能。**capex 為上游供應鏈創造數年穩定需求信號**
- **讀者洞察（Wing）**：大部分 capex 用於改造升級舊雲端架構（黃仁勳 Dwarkesh:AI 顛覆性舊資料中心全需置換;Google CEO:大部分軟體需重寫）→ 解釋 Anthropic 跑出/OpenAI 停 Sora 集中 Codex/xAI 併 Cursor。可能還有另一波軟體大清洗

## 主題六：Apple — 最好的季度遇上最大成本壓力

- **完美印鈔機**：營收 $1,112 億(+17%,史上最強三月季),iPhone $570 億(+22%),Services $310 億(毛利 76.7%,歷史新高),大中華 +28%,EPS +22%,營運現金流 >$280 億。$1,000 億回購 + 股息 +4%
- **揚棄「淨現金中性」目標**（2018 來返還 >$1 兆）→ 轉「機會導向」彈性模式（為 AI 儲備彈藥 + 潛在收購 + 新 CEO John Ternus 鋪路）
- **⚠️ 記憶體成本（最值得關注風險）**：Cook 罕見直白「六月季顯著更高,超過六月季影響越來越大」。下季毛利率指引 47.5-48.5%（本季 49.3%）。AI 伺服器/HPC 搶 DRAM/NAND → Apple 直接受影響
- **供應限制**：Mac mini/Studio 被認知為好的 AI/代理工具平台（客戶認知比預期快）+ MacBook Neo 反應 off the charts
- **CEO 交接**：Tim Cook 九月卸任,John Ternus 接（25 年職涯,延續審慎風格）。北極星「做最好產品豐富生活」
- **AI 投資增加但細節保密**（等 WWDC 六月初）。讀者:Apple 是 AI 硬體入口護城河,但須先處理落後的 Siri

## 投資 implication（接 fundamental-platform / KP 主題）

- **「變現能見度」= 跑 hyperscaler ER 的核心判準**：backlog/RPO 是 capex 去風險化的關鍵（Google $4,620億 > AWS $3,640億）。跑 GOOGL/MSFT/AMZN/META ER 時,**RPO/營收覆蓋比 + 變現路徑清晰度**比單季營收更重要 = §7.6 R7 don't fight structural（已簽長約）
- **§7.6 自我修正再驗證**：四巨頭 capex $7,250 億 + AI 營收加速 = 緩解泡沫 → 呼應 notes-41 GS「利潤奇點」+ 我系統性太保守。**這份財報是「兩引擎同時點火」的實證**（Google Cloud OM 17.8%→32.9% 翻倍 = 毛利擴張鐵證）
- **「席位+使用量」跨 SaaS 通用**（MSFT GitHub Copilot 2026/6 轉 usage）→ 接 deep-47 Cloudflare / notes-44 Snowflake / notes-41「賣工作量」。跑 SaaS ER 必問
- **記憶體成本（Meta + Apple 雙重提示）= MU ER 鐵證再加碼**：四巨頭 capex 推升 DRAM/NAND,Apple 罕見直白警告,Meta capex 上調主因記憶體 → **記憶體超級週期需求方多元（資料中心 + 消費電子雙引擎）**。接 notes-42/45 的「砍 LPDDR5X 非 HBM4」。MU 仍是最高優先 ER
- **晶片自研三雄對照**（已跑 AVGO）：AWS Trainium/Graviton（前三大資料中心晶片）+ Google TPU（對外銷售挑戰 NVIDIA）+ Meta 自研 → 跑 NVDA/AVGO ER 須把「hyperscaler 自研晶片」當長尾競爭變數,但也是 AVGO（代設計）的順風
- **ER 候選更新**：GOOGL（全棧 + TPU 外銷,最高變現能見度,九大贏家）/ AMZN（已隱含,垂直整合 + $3,640億 backlog）/ MSFT（席位+使用量 + $370億 AI 跑速）/ META（§7.6 bubble check + 信仰測試,讀者 34% 放空）/ AAPL（記憶體逆風 + WWDC catalyst,防禦 beta）
- **§7.5 訓練**：KP「不看後視鏡（追強汰弱）」= §7.5 FOMO lens 精神。「已被定價的現實」(好消息一日內反映) → 跑 ER 要問「市場織入多少樂觀/悲觀」= reverse DCF（§7.6 R6）
