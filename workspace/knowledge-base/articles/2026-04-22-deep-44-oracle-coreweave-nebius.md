# AI還是泡沬嗎？資本支出仍值得擔憂嗎？雲端反彈力度最強？- 深入分析第44期：Oracle, CoreWeave, Nebius

## Summary

API、1,000+ 家年付百萬美元客戶在兩個月內倍增——證明 AI 不是消費者玩具而是企業可付費黏性軟體；(2) Andy Jassy 5,000 字股東信首次攤牌，AWS 的 AI ARR 已超過 $15B（是當年 AWS 同期規模的近 260 倍），且明言「產能部署多快變現就多快」+ AWS ## Key Takeaways

### 敘事轉折的兩個關鍵證據（為何 AI 泡沫論已被部分化解）
- **Anthropic 4 個月 ARR $9B → $30B+**（超越 OpenAI），80% 企業端 API、1,000+ 家年付 $1M+ 客戶兩個月翻倍、Claude Code 單一條線 $25 億 ARR；訓練成本只有 OpenAI 的 1/4 + 部分業務正毛利。AI = 企業願意按月付費的高黏性嵌入式工作流軟體，不是消費者玩具
- **AWS 首次攤牌 AI ARR > $15B**（是當年 AWS 同期規模的 260 倍），Andy Jassy 明言「產能部署多快、變現就多快」+ 兩個大客戶試圖直接買下整個 2026 年 Graviton 自研晶片產能（被拒）；自研晶片業務（Graviton/Trainium/Nitro）ARR 突破 $20B
- 2026 年 hyperscaler capex 預估 $5,000-7,000 億 = 美國 GDP 1.3-1.6%，市場核心問題已從「企業會不會付錢」轉成「支出能跑多快」

### 為什麼聚焦 AI 基建層而非應用 / 晶片
- 應用模型層曝險不純（Anthropic/OpenAI 未上市，Copilot/Gemini 被埋在母公司大盤）
- 上游晶片層 NVDA/TSM **泡沫論期間沒受傷**（晶片照樣出貨）→ 完美定價、無不對稱回報
- 中間基建層在泡沫恐慌中跌最深、極端 operating leverage（固定成本極高、一旦越過損平點增量營收 60%+ 毛利率直接變淨利）→ 需求反轉時 rerating 空間最大
- 雲端三巨頭基數太大、彈性受限；本報告聚焦光譜兩端：**Oracle**（投資級混合）+ **Neoclouds**（純 AI 曝險）
- Neocloud 戰場有 ~190 家但具規模的少數，本報告不講的還有 Lambda（開發者）/ Crusoe（垂直整合能源、OpenAI Stargate 主合作）/ Nscale（歐洲主權）/ IREN（前 Iris Energy，比特幣挖礦轉 AI、$97 億微軟合約）

### Oracle — 「老富豪」的 AI 賭注
- 2025/9 公布 $5,530 億 RPO（YoY +325%）股價單日 +36%、Larry Ellison 短暫成全球首富、市值破兆
- Q4 2025 翻車：單季 capex $186 億 > 單季營收 $172 億，全年計畫 $500 億 = 全年營收 75%、總債務滾到 $1,200 億、CDS 飆破 2008、半年蒸發 $4,600 億市值
- 管理層 2-4 月救火：(1) **$50 億股債混合融資**（一半發債一半發新股，超額認購 5-6 倍 = $1,200 億搶單，保住投資級）；(2) **裁員 ~3 萬人**省 $100 億/年；(3) 換能源背景 CFO + Bloom Energy 燃料電池 55 天就能部署繞過電網排隊；(4) **BYOH（Bring Your Own Hardware）** 客戶自帶 GPU 鎖定 $290 億合約 = Oracle 從 "GPU 供應商" 變 "GPU 旅館"
- **GPU 租賃毛利率僅 ~16%**（The Information 2025/10 內部文件揭露），vs Oracle 軟體業務 70-80%；Q4 2025 出租 Blackwell 甚至虧 $100M
- 管理層目標 30-40% 但需 12-18 個月產能爬坡 — 「提升利潤率最快方法是加速容量上線」
- 三大護城河：**(A) $5,530 億 RPO**（FY26 營收 $670 億 → FY27 預計 $900 億 +34%）；**(B) 每年 $235 億軟體現金流**（金融海嘯 / 疫情都穩，提供估值地板）；**(C) 多雲資料庫**：Oracle Database@Azure/AWS/Google 已部署 70+ 區域，Q1 FY26 多雲 DB 營收 YoY **+1,529%**、目標 5 年內 $200 億——把競爭對手變成自己的分銷渠道
- 技術差異化：盒外網路虛擬化（CPU 100% 給運算）+ RDMA RoCEv2 高速互連 + 比 AWS/Azure 便宜 30%+ 全球一致定價（無境外 35% 過路費）+ 裸金屬可秒換租戶
- Larry Ellison 個人持股 > 40%（科技大廠罕見），管理層利益高度與股東一致
- 風險：FY26/27 自由現金流分別 -$240B / -$287B；客戶集中度（OpenAI Stargate 4.5GW、Meta、xAI、TikTok）；Abilene 600MW 擴建已暫停雖核心仍在
- 三情境：**樂觀 30% $280-336**（RPO 轉換 40-45% + EPS 升至 $10、P/E 28-32x）/ **基準 50% $160-255**（RPO 33% 轉換、EPS $7.6-8.2、P/E 21-23x）/ **悲觀 20% $75-140**（產能重分配給其他客戶、SOTP 軟體現金牛仍提供下檔保護）
- 催化：6 月中 Q4 FY26 財報是近期最重要 single event；Bloom Energy 部署實際 GW 數；Stargate / OpenAI 合約上線進度

### CoreWeave — 「天才交易員」的極端槓桿
- 起源：2017 年三位前大宗商品交易員撞球桌挖礦 → 2018 礦難轉型雲端 → ChatGPT 浪潮接力，三年從 $16M 衝到 $5.13B
- 商業模式：純 GPU 雲端、比 hyperscaler 便宜 30-60%、裸金屬 Kubernetes + InfiniBand、捨棄虛擬機
- **NVIDIA 共生關鍵**：CoreWeave 是 NVIDIA 最大 GPU 雲端客戶中**唯一沒在開發競爭性矽晶片**（AWS Trainium / Google TPU / MS Maia / Meta MTIA 都在自研）→ 優先拿到 GB300 + NVIDIA 1 月再注資 $20 億
- 4 月轉折：股價單月 +50%。Meta 擴約至 $350 億總承諾（合約至 2032）、Anthropic 多年合約、Jane Street $10 億入股 + $60 億算力承諾。CEO 強調**單一客戶營收 < 35%**（Microsoft 從 67% 降下來）
- 資本市場升級：1.5-2% 可轉債（vs 2024 的 11% 高息擔保），定價從「投機性高息垃圾債」轉「成長型企業債」
- 但 Q1 2026 指引 $1.9-2.0B 低於 $2.29B 預期 — 揭露 Neocloud 模型核心矛盾：**backlog ≠ 立即可認列營收**（基礎設施還在建、還沒通電、客戶還沒上線）
- **會計與槓桿警訊（投資人必看）**：
  - 營收成本**不含** GPU 折舊（藏在「技術與基礎設施」混雜研發等），財報毛利率被「美化」。Oracle 同類業務目標毛利率 35%，CoreWeave 表面「漂亮」可能不真實
  - GPU **折舊用 6 年** vs Nebius 4-5 年（會計選擇影響毛利率呈現）
  - $300 億帳面債務 + **$340 億 lease 義務 = $640 億真實負擔**，是 2025 年營收 $51B 的 12.5x
  - **$210+ 億擔保貸款用 GPU 抵押** — 若 GPU 貶值快過 6 年折舊曲線 → 抵押品覆蓋率惡化 → 經典「死亡螺旋」
  - 利息費用 / 營收 ~25%；Q4 2025 利息 $388M 直接導致 $452M 淨虧
- 管理層反駁：「齒輪箱」設計每筆貸款對應一份客戶合約、上線後產生現金流還貸；2029 年前無近期到期；2025 籌資 $180 億後加權利率降 300 bps
- 但對比 Meta 每年 $1,500 億廣告營收支撐，CoreWeave 用 $51B 撐 $300-350 億 capex 是 6x 槓桿
- 三情境：**樂觀 25% $175-295**（DA Davidson $175、Arete $295）/ **基準 50% ~$126**（共識）/ **悲觀 25% $38-99**（MS $99、HSBC $38）
- 催化：5 月 Q1 財報能否 beat $1.9-2.0B 保守指引；Vera Rubin 部署進度；利息 / 營收 比能否從 25% 降至 < 15%

### Nebius — 「戰場老兵」的歐洲牌
- 1997 年俄羅斯 Yandex（巔峰市值 $300 億）2022 年制裁停牌 → 2024 創辦人 Volozh 以 $54 億剝離俄業務 + 保留雲端 + 數據中心 → 2024/10 重返 NASDAQ 改名 Nebius
- 「不是空殼重生」：1,300 員工中 1,000 名前 Yandex 工程師（vs CoreWeave 金融背景），有從數據中心 / 主機板 / 伺服器 / 網路全端自研能力
- **戰略資產包**（隱藏 cash bank）：ClickHouse $150 億股權、Avride 自駕（Uber 合作 $3.75 億）、TripleTen 教育
- 2025 Q4：營收 $228M YoY **+547%**、AI 雲端 **+830%**、毛利率 70%、ARR 達 $1.25B、新簽合約 Q3→Q4 翻倍 + ASP 逆勢漲 50%+
- **Backlog $450-500B vs 2025 營收 $5.3B = 比例 94:1**（極端）
- 微軟 9 月簽 $194 億最高合約（10 萬顆 GB300） + Meta 擴約至最高 $270 億合約
- NVIDIA 累計注資 ~$27 億（含 3 月最新 $20 億取得 8.3% 股權），目標 2030 前部署 5GW
- **歐洲主權 AI 不可替代**（GDPR / AI Act）— 美國 hyperscaler 進不去，這是 CoreWeave 沒有的牌
- 基礎設施：芬蘭 40MW 自建 + EMEA 鎖定 750MW 電力 + 美國密蘇里州 1GW 擴張中
- **資本結構是最大優勢**：2026 capex $16-20B 中 **60%+ 由客戶預付款 + 既有現金 + OCF 覆蓋**，剩 40% 才靠融資；3 月 $43.4 億可轉債（1.25% / 2031 + 2.625% / 2033 超額認購並上修）；遞延收入 $15.8 億（微軟/Meta 預付）+ 現金 $36.8 億；總債務 $85 億 << CoreWeave 的 $300 億
- 風險：(A) capex 是現營收 30-40 倍 + 微軟 / Meta 高度集中；(B) 從零負債公司轉變為管理數百億複雜資本結構是新手考驗；(C) **EV $410 億 / TTM 營收 $5.3B = P/S 76x**（vs CoreWeave 12x），完美定價極低容錯率
- 三情境：**樂觀 25% $205+**（Goldman 目標 + 30-50% 上行）/ **基準 50% ~$176**（共識）/ **悲觀 25% $85**（Morningstar 最低）
- 估值差異解讀：Nebius EV/前瞻營收 10.8-15.6x vs CoreWeave 5.2-8.5x 不是因為 Nebius 「貴」，而是 (i) 較小規模 + 較快增速給溢價；(ii) 全棧自研軟體加分；(iii) 客戶預付款結構降低風險獲得 rerating
- 催化：5/18 Q1 財報；微軟合約認列進度 + capex 預付款是否真覆蓋 60%；Vera Rubin 部署；年底 ARR 70-90 億達標

### 三家公司「賭法」對比 + 投資決策框架
- **Oracle = 老富豪**：年度 capex 逼近全年營收，但即使輸還有每年 $235 億軟體現金流回家吃飯；提供確定的下檔保護 + 溫和上行
- **CoreWeave = 天才前交易員**：刷 11% 利率信用卡買籌碼，三年翻 300 倍；牌技驚人但時間成奢侈品；要麼大贏要麼大輸
- **Nebius = 戰場老兵 + 智者**：說服牌桌最大兩玩家（微軟/Meta）先幫付籌碼；牌面最乾淨但建設與供應鏈執行風險巨大
- **市場定價**：Oracle = 轉型中軟體巨頭（軟體現金牛當 floor）、CoreWeave = 高成長高槓桿基建（market 在懷疑 balance sheet）、Nebius = 公開市場 venture（為 2028 完美執行提前付帳）
- **決策四步**：(1) 你對 AI 基建需求信念多強？(2) 你的真實風險承受能力（不是「以為」而是腰斬時實際會做什麼）？(3) 「如果我錯了」 stress test：Oracle 安全墊 = $235B 軟體現金流、CoreWeave 安全墊「幾乎沒有」、Nebius 安全墊 = $36.8B 現金 + ClickHouse 等非核心資產（阻止災難不阻止大幅回撤）；(4) 考慮組合配置而非單一押注
- **單選題**：怕錯過 → CoreWeave / Nebius（做好失去 50%+ 本金準備）；怕大虧 → Oracle
- **2000 年思科教訓**：思科確實是網際網路基建，但 2000/3 買入持有 25 年仍負回報——不是技術假，是**價格太高**。**好公司 ≠ 好投資；好投資 = 好公司 × 合理價格 × 安全邊際**
